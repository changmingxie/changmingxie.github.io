注册中心 #

server模式下，client端与server端需要建立长连接进行通信。
涉及如何发现server集群的访问地址问题，这里使用注册中心来解决。

已支持注册中心 #

当前支持注册方式有：

  • direct
  • nacos
  • zookeeper
  • custom
